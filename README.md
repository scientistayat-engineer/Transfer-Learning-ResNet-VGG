# Transfer-Learning-ResNet-VGG
Cat vs Dog Image Classification using pre-trained VGG16/ResNet50 backbones and a custom styled Gradio interface.

# 🛸 Transfer Learning (ResNet/VGG) Deep Vision Analytics

An end-to-end Computer Vision inference pipeline designed to process image tensors and classify them into **Cat vs Dog Contexts** using pre-trained deep convolutional neural networks.

## 🚀 Key Features
* **Dual-Engine Core:** Feature extraction powered by frozen weights of pre-trained models.
* **Dynamic Diagnostics:** Embedded loss and accuracy tracking matrices to analyze convergence.
* **Premium Gradio Interface:** A custom emerald-themed dark user interface equipped with continuous confidence filtering sliders and real-time inference latency telemetry.

## 📦 Tech Stack & Core Dependencies
* **Framework:** TensorFlow / Keras
* **Interface:** Gradio
* **Data & Analytics:** NumPy, Matplotlib

## 🔧 Deployment Layout
The workspace is structured seamlessly across logical execution blocks:
1. **Phase 1 & 2:** Environment Initialization & Freezing Base Model Weights.
2. **Phase 3 & 4:** Custom Top-Head Assembly & Dynamic Callback Compilations.
3. **Phase 5 & 6:** Performance Curve Plotting & Live Premium UI Node Deployment.
